NYC Apartment Rent Prediction & Analysis

About:

Here you will find tools to predict the monthly rent of an apartment in Manhattan, Queens, or Brooklyn depending on the features you choose. This analysis is based on listings for approx. 5,000 available apartments. (Source: kaggle.com)

Main Features:

Rent Forecasters:


Optimal Forcast Model

Enter values for the the apartment features and get a rent prediction based on an optimized machine learning model.

Custom Forcast Models

Select from a list of up to 17 apartment features and get a rent prediction based on multiple models that are generated on-demand based on your selections, and inputs.

Visual Analysis

An in-depth analysis of apartment listings with interesting visualizations.
